# Supplier confirmation rules (anchor company)

Use these rules whenever you label a company as a supplier to the **anchor company** in your report.

## Allowed labels

- **Anchor-confirmed**: backed by at least one high-quality primary source (see below).
- **Not confirmed**: credible evidence suggests it is *not* a supplier, or it is only a generic category supplier with no program tie.
- **Rumor**: there are recurring market claims it is a supplier, but you do **not** have primary-source confirmation.
- **Unknown**: anything else (most cases).

## What counts as “Anchor-confirmed” (preferred → acceptable)

1) Anchor company primary sources
- Anchor company SEC filings naming the supplier
- Anchor company IR decks / official blog / press releases naming the supplier

2) Supplier primary sources
- Supplier SEC filing / annual report explicitly naming the anchor company and the relevant program/category, with dates

3) Joint confirmation
- A joint press release or two-sided announcement where both the anchor company and the supplier are named

## What does NOT count

- Single-source media claims without primary-source backing
- “Supply chain research says X%” without citations to filings/official documents
- Teardowns with unclear provenance

## When to use “Rumor” (and how to write it)

Use **Rumor** only when all are true:
- You have **≥2 independent** non-primary sources making the supplier claim (e.g., multiple media outlets / multiple sell-side notes / multiple event recaps).
- The claim is **specific** (which program/category, timeframe, and what is supplied), not vague “in the supply chain”.
- You clearly state it is **unverified** and list what primary evidence would upgrade it to **Anchor-confirmed**.

Minimum note standard for **Rumor**:
- list the 2+ sources (URLs or document paths) with dates
- add a short “upgrade path” line: which filing/PR/IR doc would confirm or refute

## Minimum evidence note standard

When you mark **Anchor-confirmed**, include:
- source type (Anchor filing / Supplier filing / Joint PR)
- publication date
- a URL or document path
