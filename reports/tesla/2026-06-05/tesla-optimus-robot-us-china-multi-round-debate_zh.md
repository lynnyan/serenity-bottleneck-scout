## 0) 范围、定义与假设

本报告做什么：
- 构建一个覆盖 **美国 + 中国** 生态的、**端到端的实体供应链地图**（从原料到部署）。
- 只选择 **一个“被研究节点（node under study）”**，并给出可量化的 bottleneck 测试口径（即使目前缺数据）。
- 严格执行 **供应商确认规则**（没有主证据就不标注 Anchor-confirmed）。

本报告不做什么：
- 不提供个性化投资建议、仓位或交易时点。
- 不声称 Tesla 已经实现大规模人形机器人出货（本轮未提供可验证的出货量证据）。

定义：
- **Chokepoint（扼喉点）**：多个下游生态都绕不开的“必经关口”（标准/平台/接口/认证门槛等）。
- **Bottleneck（瓶颈）**：供给扩产慢且可量化，导致交期/交付/定价被少数供给方“卡住”。

关键假设（明确标注）：
- 若人形机器人规模化，其核心从“demo”走向“量产”，瓶颈更可能出现在执行器可靠性、安全合规、以及制造吞吐/良率学习曲线。
- 本轮引用的 Tesla 披露属于 **意图/产能口径（intent/capacity framing）**，不等于已验证的外部需求/出货爬坡。

哪些信息会让我改变观点：
- **更乐观**：Tesla（或其他 Tier‑1）披露带日期的 **部署量/运行速率/商业化里程碑**；供应商在财报中披露与该节点相关、可归因的 backlog/capex。
- **更悲观**：连续多个季度缺乏任何数值化爬坡信息；设计路线变更导致单机“该节点含量”下降；快速多供/垂直整合导致供应商价值捕获消失。

## 1) 供应链地图（实体、端到端）

原材料 → 零部件 → 模组 → 整机系统 → 部署与运维

1) **原材料**
- 铜（绕组）、铝（结构件）、特种钢
- **电工钢/硅钢**（电机叠片核心材料）
- **稀土永磁 NdFeB**（高扭矩密度电机关键材料）
- 弹性体/胶黏剂、导热界面材料、润滑材料

2) **零部件（机器人 BOM 基元）**
- 电机核心（叠片 + 定转子制造 + 绕组 + 绝缘/浸漆 + 品控）
- 功率电子（逆变/伺服驱动、栅极驱动、传感、电容等）
- 精密传动（减速器/齿轮/丝杠）、轴承
- 传感（编码器/旋变、力矩/力传感、IMU）、视觉计算与相机
- 线束/连接器（高弯折）、制动器（失电保持）

3) **模组**
- **关节执行器模组**（电机 + 传动 + 功率电子 + 传感 + 热管理 + 壳体）
- 手部/夹爪（微型齿轮、腱绳、触觉阵列）、前臂组件
- 电池/供电模组

4) **系统集成**
- 整机装配 + 标定 + 端检
- 工业现场安全与功能安全（workplace integration）

5) **部署与运维**
- 美国：工厂部署、EHS/安全集成、服务与备件体系
- 中国：工厂部署 + 本地合规 + 国产替代压力

“必经关口”（假设，待证据）：
- 工业现场的安全/功能安全合规体系（关口存在，但“该节点是否因此形成供应商优势”需证明）。
- 高磨损模组（执行器/线束）随着在役数据积累带来的再验证成本。

潜在“实体供给约束点”（假设，待证据）：
- 高牌号电工钢供给 + 叠片产能/良率。
- NdFeB 永磁材料的质量一致性与配额分配。
- 精密制造与计量能力（磨削/齿形加工/三坐标等）作为隐藏瓶颈。

## 2) 被研究节点（只选一个）：关节用“高扭矩密度电机核心制造”

节点定义（到底研究什么）：
- 人形机器人关节执行器的 **电机核心制造链条**：电工钢叠片 → 定转子加工 → 绕组（如发卡/圆线）→ 绝缘/浸漆 → 动平衡/品控。
- 该节点嵌入执行器模组，随机器人数量线性放大，并深度影响可靠性与成本曲线。

为什么选这个节点（map-first 的理由）：
- 若 Tesla 提到的 100 万/年、1,000 万/年产线设计在未来“方向上接近真实”，电机核心制造必然面临 **吞吐 + 良率学习** 的挑战。
- 真正的瓶颈可能来自“隐藏层”：不只是材料短缺，更是合格产能、计量体系、在役失效学习闭环。

替代路径（对节点价值捕获不利）：
- 设计简化：降低扭矩密度要求（更慢、更少自由度/任务），允许低规格电机。
- 垂直整合：Tesla 或中国整机厂自建电机核心/执行器能力。
- 执行路线变化：传动方案改变导致单机电机核心强度下降。

“不可绕开（unavoidable）”说明（明确假设 + 证伪条件）：
- 假设：近中期主流人形方案以 **电驱关节执行器** 为主（电机 + 功率电子 + 传感），而非多数自由度使用液压/气动。
- 在该假设下，**电机核心制造不可绕开**：任何电驱关节都必须有电机核心。
- 证伪：可信披露/拆解显示主要执行方式并非电驱，或设计显著降低单机电机核心含量。

可量化的 bottleneck 测试口径（用于“证伪/证实”）：
- **交期**：相关产能/设备/关键材料交期持续 >12 个月。
- **Backlog**：供应商披露与机器人/人形相关的明确 backlog（而非泛化“自动化”）。
- **Capex**：公开披露为叠片/绕组/浸漆/品控扩产的资本开支，且投产周期长。
- **良率/质量**：财报或披露出现与执行器/电机相关的质量/召回/保修压力。
- **价格/结构**：高牌号电工钢或磁材出现持续的结构性涨价/紧缺信号。

本轮证据缺口（当前）：
- 没有主证据确认 Optimus 的电机核心上游供应商。
- 没有纳入任何可复核的交期/backlog/capex 数据集。

如何补齐证据缺口（最小可行验证计划）：
- **财报关键段落抽取（risk/backlog/capex/geo）**：从主文件中抓取可复核段落并持续跟踪。

## 7) 风险与可量化反证（明确）

- 需求不确认：连续多个季度仍无任何数值化爬坡/部署披露。

- 地缘政策：出口管制/合规变化改变可用供应商集合；中国国产替代节奏加快。
