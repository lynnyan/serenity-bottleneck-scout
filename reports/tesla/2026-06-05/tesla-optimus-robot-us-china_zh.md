## 0）范围与假设

- 锚定公司/需求：Tesla（Optimus 人形机器人）。
- 覆盖范围：
  - 识别 Optimus 规模化部署最可能遇到的**物理门槛**。
  - 建立美国+中国上市公司“组件优先”候选池（不是已确认供应商名单）。
  - 区分 **chokepoint**（必经接口/标准/认证门槛）与 **bottleneck**（可量化的供给约束）。
- 不覆盖：
  - 未核实的“供应商名单”当作事实。
  - 个性化仓位配置与买卖建议。
- 关键假设：
  - 1–3 年不确定性更多来自“可制造性 + 可靠性 + 安全部署约束”，而不是“AI 训练算力”本身。
  - 若 Tesla 仅有愿景/设计产能表述、没有单位数/产量/部署数，则本研报定位为“地图优先”，而非“瓶颈已证明”。
- 可能改变结论的证据：
  - Tesla 给出时间绑定的数字披露（部署数、产量/生产率、产线投产里程碑），从而确认或否定近端 ramp。
  - 核心关节部件二供/三供认证很快，且再验证周期短。
  - 架构变化降低对减速器/丝杠等精密部件依赖，并有可信现场可靠性验证。

## 0.1）研究节点（只选一个）

**研究节点（bottleneck 候选）：**可规模化的**关节模组可制造性**（电机 + 减速/丝杠 + 编码器 + 制动器 + 热 + 线束），在目标寿命、安全边界与成本下实现可复制的良率与一致性。

理由：
- 人形机器人要成为“物理业务”，首先要能把高自由度关节模组稳定量产并控制现场失效率。
- 当前这是 bottleneck 的**假设**，必须用 lead time/backlog/capex/yield 等硬证据验证。

## 1）供应链地图（物理闭环）

原料 → 零部件 → 模组 → 系统 → 部署

- 原料：
  - 稀土（NdPr；高温抗退磁的 Dy/Tb 等）→ 钕铁硼磁体
  - 铜（绕组）、高牌号电工钢叠片、轴承/齿轮钢
  - 聚合物/弹性体（线缆绝缘、密封）、胶黏剂、导热界面材料
- 零部件：
  - 执行：无框力矩电机/BLDC；减速器（谐波/摆线/RV）；滚珠丝杠/线性执行器；轴承；制动器
  - 反馈：绝对值编码器；扭矩/力传感；IMU；（早期可选）触觉阵列
  - 电源：电芯+BMS；DC/DC；逆变器；关键被动件；热管理（泵/阀/冷板）
  - 互连：高柔线束；连接器；滑环/拖链
  - 感知/计算：摄像头；（可选）LiDAR/3D；计算模组；安全计算路径
- 模组：
  - 关节模组（电机 + 减速/丝杠 + 编码器 + 制动器 + 热 + 线束）
  - 手/末端执行器（微型齿轮/轴承 + 触觉 + 电机）
  - 移动底座 + 电源分配
- 系统：
  - Optimus → 工厂内部部署 → 潜在外部部署

需要证明的 chokepoint（不能默认成立）：
- 安全/认证/合规门槛：必须命名适用标准/认证路径与再验证机制。
- 关节模组制造门槛：精密加工 + 计量检测 + 装配良率学习曲线。

需要测量的 bottleneck（不能默认成立）：
- 关节模组关键部件的交期/积压/扩产/良率约束。
- 现场可靠性约束（线束、热、制动器磨损等）导致 ramp 放缓。

地理/政策：
- 美国产线 ramp 与“中国产业链参与度”的拉扯 → 关税/出口管制 + 再认证风险。
- 稀土加工集中度 → 磁材供给政策冲击。

## 5）催化剂时钟（含 go/no-go）

硬催化（3–9 个月）：
- Tesla 接下来 1–2 个报告窗口：是否出现**数字化**披露（部署数、产量/生产率、产线投产里程碑）。
- 任意供应商财报/双方 PR 明确点名 Tesla + robotics/actuation 项目（必须一手）。

确认催化（12–18 个月）：
- 供应商财务中出现可归因于机器人项目的 backlog/收入/毛利变化。
- >12–18 个月交期持续（支持 bottleneck）或快速多供方认证（反证 bottleneck）。
- 内部部署规模可观察扩大（超出原型），间接反映良率可复制。

Go/no-go：
- 到 Tesla 接下来两次更新窗口仍无数字披露：下调“需求确认”强度，定位长期观察。
- 到识别出具体供应商后的两次报告窗口仍无积压/扩产/良率证据：将价值捕获降级为期权价值。
