$绿的谐波(SH688017)$  

截至2025年底，绿的谐波与国内机器人厂商及特斯拉的合作进展如下：


一、与国内机器人厂商的合作

1. 宇树科技

绿的谐波为其人形机器人提供谐波减速器，双方合作已落地并实现批量供货。

2. 优必选

作为核心零部件供应商，绿的谐波参与优必选人形机器人关节模组的开发，2025年订单量显著增长。

3. 其他国产机器人企业

公司公开表示与国内多家机器人客户“保持良好合作关系”，涵盖工业、服务及特种机器人领域。

二、与特斯拉的合作

1. Optimus人形机器人

- 供应商地位：绿的谐波已进入特斯拉Optimus供应链，成为其谐波减速器二级供应商。

- 订单规模：2025年签订1万套谐波减速器订单，并于当年第二季度完成交付。

- 技术适配：为Optimus开发Y系列超高精度减速器（精度±10角秒），解决关节过热与噪音问题。

2. 合资扩产

与三花智控在墨西哥设立合资公司，规划年产能50万套执行器，预计2027年占特斯拉供应链份额超50%。

3. 未来展望

特斯拉计划2026年量产第三代人形机器人，年产能目标100万台，绿的谐波作为核心供应商将直接受益。

总结

绿的谐波已成为国产机器人产业链核心配套商，并深度绑定特斯拉Optimus，随着人形机器人进入放量期，其供应链地位有望进一步提升。

绿的谐波（688017.SH）作为国产谐波减速器龙头，已与全球工业/协作/人形机器人巨头建立稳定合作与量产/送样，截至2025.12公开口径，核心清单与进展如下。


国际合作（工业/协作/人形）


- Universal Robots（UR）：2014年送样、2016年签长期框架，持续批量供货协作机器人谐波减速器，全球化标杆客户。


- Tesla Optimus：进入人形机器人供应链，谐波减速器用于关节传动；与三花智控墨西哥合资建厂，支撑北美量产与交付，是关键增量来源。


- Figure AI：人形机器人核心供应商，已获订单/送样，受益其量产落地。


- ABB/发那科/那智不二越等：精密零部件与谐波减速器配套，覆盖工业机器人，推进国产替代与规模化供货。


- 斗山/Rainbow Robotics（韩国）：通过经销商 Baum Newtec 间接配套协作机器人关节模组。


国内合作（人形/工业/协作）


- 优必选：Walker系列等核心谐波减速器供应商，人形关节驱动关键件，深度绑定并量产。


- 智元机器人：小批量/量产阶段配套，适配人形机型关节，订单与出货同步提升 。


- 小鹏机器人 IRON：手部关节等谐波减速器供应，超小型高精度型号适配灵巧手。


- 新松/埃斯顿/埃夫特/华数/新时达/亿嘉和/广州数控/配天技术等：工业/协作机器人批量配套，国产替代主力。


- 华为/宇树科技/小米 CyberOne/达闼科技等：送样或定点，覆盖人形与特种机器人，加速国产供应链闭环。


关键合作载体与催化


- 墨西哥合资（与三花智控）：服务全球人形客户，缩短交期、规避贸易壁垒，支撑北美量产落地。


- 催化节点：量产订单落地、人形机型批量交付、海外认证通过、产能爬坡与良率提升。


风险提示
